# SaveWise UI

SaveWise UI is a reusable component library for the SaveWise platform.

It provides consistent, accessible, and composable interface building blocks that can be used across the SaveWise frontend applications.

## Features

- Buttons, inputs, and form controls
- Cards, modals, and alerts
- Tables and layout primitives
- Dashboard widgets
- Progress and status components
- Accessible interaction patterns
- Shared design consistency

## Tech Stack

- React
- Component styling approach of choice
- Accessibility-first design patterns

## Getting Started

### 1. Clone the repository
```bash
git clone <your-savewise-ui-repo-url>
cd savewise-ui
```
### 2. Install dependencies
```bash
npm install
```
### 3. Start local development
```bash
npm run dev
```
### Scripts
```bash
npm run dev
npm run build
npm run lint
```

## Purpose
This repository promotes design consistency, reuse, and accessibility across the SaveWise frontend ecosystem.
